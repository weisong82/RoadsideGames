# 🌟 路上小游戏

> 和孩子一起发现路上的美好

一套专为亲子互动设计的移动端网页小游戏，让散步、上学路上的时光变得更有趣。无需安装，直接在浏览器中打开即可使用。

## 游戏列表

| 游戏 | 描述 | 解锁徽章 |
|------|------|---------|
| 🏪 猜猜这家店开了几年 | 路过小店时，和老板聊聊店里的故事 | 小店侦探 |
| 🌿 认认路边的植物 | 一起看看梧桐叶、樟树叶长什么样 | 植物学家 |
| 🎵 声音小游戏 | 闭上眼睛听 30 秒，听听路上有什么声音 | 声音猎人 |
| 🎨 色彩寻宝 | 边走边找红色、黄色、绿色 | 色彩探险家 |
| 👋 看看身边的人 | 和快递员、环卫工人聊聊天 | 社交达人 |

## 功能特性

- **每日推荐**：每天根据日期自动推荐一款游戏
- **步骤引导**：每款游戏分步骤引导，包含观察、互动、挑战等环节
- **徽章系统**：完成游戏 3 次即可解锁对应徽章，共 7 枚可收集
- **连续打卡**：记录连续游玩天数，培养习惯
- **数据持久化**：游戏进度和徽章保存在本地，无需账号

## 快速开始

本项目无需构建工具，直接用静态服务器托管即可。

```bash
# 使用 Python
python3 -m http.server 8080

# 或使用 Node.js
npx serve .
```

然后在浏览器中打开 `http://localhost:8080`。

> 也可以直接在浏览器中打开 `index.html`，但部分浏览器对 `file://` 协议下的 `localStorage` 有限制。

## 怎么玩

1. 打开应用，查看当天的"今日推荐"游戏
2. 或者从游戏列表中自由选择
3. 按照步骤引导，和孩子一起完成观察与互动任务
4. 完成游戏后收集徽章，记录探索足迹

## 项目结构

```
index.html              # 应用入口，底部导航，脚本加载顺序
├── css/
│   └── main.css        # 全局样式，CSS 自定义属性设计令牌
├── js/
│   ├── store.js        # 状态管理（localStorage 持久化、徽章、连续天数）
│   ├── games.js        # 游戏注册表与元数据，每日游戏选择逻辑
│   ├── game-engine.js  # 步骤式游戏运行器
│   └── app.js          # 路由与页面渲染（首页、徽章、关于、游戏）
├── games/
│   ├── shop-game.js    # 猜猜这家店开了几年
│   ├── plant-game.js   # 认认路边的植物
│   ├── sound-game.js   # 声音小游戏
│   ├── color-game.js   # 色彩寻宝
│   └── people-game.js  # 看看身边的人
└── components/
    └── badge-card.js   # 徽章卡片 HTML 生成器
```

## 技术说明

- **纯原生技术栈**：原生 JS / CSS / HTML，无框架，无构建工具
- **全局模块导出**：各模块挂载到 `window`（如 `window.Store`、`window.Games`），脚本加载顺序即为依赖顺序
- **哈希路由**：`#/`、`#/badges`、`#/about`、`#/game/:id`
- **步骤类型**：`intro` → `observation` → `timer` → `input` → `challenge` → `complete`
